Information about Reserve My Show


General Information

This web application is for the reservation/booking system for a theatre. The system implements the following functionality:

- Users can browse the different seats available
- Seats can be searched/filtered by:
  • The date/time for the booking
  • The seating section


How to use the website?

On the top-right, there are three button:
  • Book a Show: Lets the user choose from the movies available and the user can choose a movie to watch from a given date/time
  • Change Content: Changes the content of the website to display more informaiton about the website or show the movies available
  • Dark/Light: Lets the user choose the light or dark mode, according to their preference


How to select a movie?

Click on 'Book a Show' on the top right button and choose from the given movies.
Then, choose from the available dates and times along with the seating section for the movie.
Finally, click the submit button. A confirmation message should appear as an alert!
